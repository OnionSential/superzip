SuperZip — что куда положить
==============================

Структура папки (положи всё это в свой проект ровно так):

SuperZip/
├── sz.c                      движок №1: LZ77 + Хаффман, уровни 1-9
├── mix.c                     движок №2: контекстный смеситель + SSE (жмёт сильнее всего)
├── superzip_app.html         живой макет интерфейса — открывается в браузере,
│                              просто для справки/дизайна, в проект НЕ компилируется
└── installer/                всё для сборки установщика (как у WinRAR)
    ├── superzip_setup.iss    сам скрипт установщика
    └── assets/
        ├── app_icon.ico      иконка приложения
        ├── wizard_banner.bmp баннер слева в окне установки
        └── wizard_small.bmp  маленькая иконка в углу окна установки


1) ДВИЖКИ (sz.c и mix.c) — твоя основная работа
--------------------------------------------------
Это два рабочих архиватора, написанных с нуля. Собираются и проверяются
одинаково, через терминал:

    gcc -O2 -o sz.exe  sz.c
    gcc -O2 -o mix.exe mix.c

Сжать файл:
    .\sz.exe  c file.txt file.sz 9      (9 = максимальный уровень)
    .\mix.exe c file.txt file.mx        (у mix.exe уровней нет — он один, и сильный)

Распаковать:
    .\sz.exe  d file.sz  back.txt
    .\mix.exe d file.mx  back.txt

Если не уверен, какой использовать: sz.exe — быстрее, mix.exe — жмёт плотнее,
но медленнее. На больших текстовых файлах mix.exe обходит даже gzip.


2) superzip_app.html — дизайн-макет
--------------------------------------------------
Это НЕ часть программы, а кликабельный прототип того, как должны выглядеть
окно сжатия и мастер установки. Просто открой файл в браузере (двойной клик).
Он показывает идею для будущего GUI (на Win32/Qt), а в реальный exe пока
не превращён — это макет, не код установщика.


3) installer/ — сборка настоящего установщика
--------------------------------------------------
Тут получится файл вроде "WinRAR-инсталлятора", но для SuperZip.

Шаги:
  а) Собери sz.exe и mix.exe (см. пункт 1) и положи ИХ ОБА прямо в папку
     installer/ — рядом с superzip_setup.iss. Это важно: установщик ищет
     файлы рядом со своим скриптом.

  б) Поставь Inno Setup (бесплатно): https://jrsoftware.org/isinfo.php

  в) Открой installer/superzip_setup.iss в Inno Setup Compiler.
     Нажми F9 (или Build → Compile).

  г) Готовый установщик появится в installer/Output/SuperZip-Setup.exe —
     это и есть финальный файл, который можно раздавать и запускать как
     обычный установщик Windows.

Что сделает установщик при запуске:
  - поставит sz.exe и mix.exe в Program Files\SuperZip
  - создаст ярлык в меню «Пуск» и, по желанию, на рабочем столе
  - по желанию добавит пункт "Сжать в SuperZip" в контекстное меню
    правой кнопки мыши (на основе sz.exe, уровень 9)
  - создаст деинсталлятор


Что НЕ включено (и почему)
--------------------------------------------------
Черновые версии lz.c (LZ77 без Хаффмана) и cm.c (контекстная модель без
модели совпадений) в пакет не вошли — это были учебные шаги по дороге к
sz.c и mix.c, и они полностью устарели. Если нужны для истории/сравнения —
просто попроси, пришлю отдельно.
